# De l'usage des deepfakes dans la guerre informationnelle moderne

*Travail pour le **CIEDS** (Centre Interdisciplinaire d'Études pour la Défense et la Sécurité), dans le cadre de l'Entente Cordiale Day Challenge, École Polytechnique.*

**Auteurs :** Arthur Filippi, Salvador Levy, Pierre Blusseau-Bourcier.

## Sujet

Le rapport (1) dresse un **état de l'art géostratégique** de l'usage des deepfakes dans plusieurs conflits récents (Russie-Ukraine, Israël-Hamas, Inde-Pakistan, Syrie, Afrique subsaharienne) ; (2) présente leur **génération** (GANs, diffusion, synthèse vocale), **détection** et les dispositifs de lutte ; (3) propose des **solutions** : *détecter* (CAPTCHA participatifs, blockchain), *sanctionner* (non-prolifération, licences d'IA), *éduquer* ; (4) examine la bascule vers un **usage offensif** par les États.
